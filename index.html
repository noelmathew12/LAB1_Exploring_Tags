Paragraph tag- This is a paragraph made using the Paragraph Tag

Bold Tag- It is used to change the text to bold.

Strong Tag- Strong Tag defines important text

Superscript Tag- It is used to Superscript the specified text

Subscript Tag- It is used to Subscript the specified text

I tag- It is used to italize the content inside the tag

Mark Tag- It is used to highlight or mark important text.

Del Tag- It is used to define deleted text.

S Tag- It is used to mark up text that is no longer correct

Details and Summary Tag

The "details" tag specifies additional details that the user can open and close on demand. The "summary tag" defines a visible heading for the "details" element. The heading can be clicked to view/hide the details.

Ordered Lists- The "ol" tag defines an ordered list. An ordered list can be numerical or alphabetical.

  1. Coffee
  2. Tea
  3. Milk

Unordered Lists- The "ul" tag defines an unordered list. An unordered list is usually in bullets.

  • Coffee
  • Tea
  • Milk

q Tag- It is used to insert quotation marks.

Samp Tag- It is used to define sample output from a computer program.

Span Tag- It is an inline container used to mark up a part of the text.



HTML Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.







The progress tag represents the completion progress of a task.

The button tag defines a clickable button.



The hr element is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page.



The footer tag defines a footer for a document or section.
